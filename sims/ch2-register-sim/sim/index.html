Instruction Cheat Sheet

            
    
                
  • MOV #5, W0: Load literal into W0
    • 
                
  • MOV W0, W1: Copy W0 into W1
    • 
                
  • ADD W0, W1: W1 = W1 + W0
    • 
                
  • ADD #5, W2: W2 = W2 + 5
    • 
                
  • ADD W0, W1, W2: W2 = W0 + W1
    • 
                
  • SUB W1, W0: W0 = W0 - W1
    • 
                
  • SUB W1, W0, W3: W3 = W0 - W1
    • 
                
  • CLR W2: Clear W2
